## Date Metadata Caveat

Many Hebrew-language articles display corrupted publication dates in Readwise (e.g., "Oct 5th 360") due to the software incorrectly parsing Hebrew calendar dates. Articles in this collection have been dated by:

1. Manual verification against article content where possible
2. Readwise `saved_date` as a proxy where `published_date` is corrupted
3. Content context (e.g., "this morning" in day-of articles)

The `day_of_oct7.csv` file contains manually verified `pub_date` values for the 26 identified day-of articles.
